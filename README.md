# next-passport-oauth-google-email-nedb

This example combines:

- [Next.js](https://github.com/zeit/next.js/)
- [NeDB](https://github.com/louischatriot/nedb)
- [OAuth 2.0](https://oauth.net/2/)
- [Passport](http://www.passportjs.org/)
